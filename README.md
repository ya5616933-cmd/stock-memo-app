# 🎯 台股記憶閃卡 MVP - 使用說明

## 快速開始

1. **開啟應用程式**
   - 用瀏覽器開啟 `index.html`
   - 建議使用 Chrome, Edge, 或 Firefox

2. **開始學習**
   - 看到股票代號後，嘗試回想公司名稱與業務
   - 點擊卡片翻轉查看答案
   - 根據您的記憶狀況選擇按鈕

## 功能說明

### 📊 儀表板
- **已精通**: 連續答對 3 次以上，進入長期記憶
- **學習中**: 正在學習的股票
- **太難 (封存)**: 放棄的股票，不再出現
- **待學習**: 尚未看過的新卡片

### 🎴 閃卡操作
- **正面**: 顯示股票代號
- **背面**: 顯示公司名稱、產業、業務描述
- **點擊卡片**: 翻轉查看答案

### 🎛️ 三個按鈕
1. **太難了 / 非能力圈** 🚫
   - 這家公司太複雜，直接封存
   - 符合巴菲特「能力圈」概念
   - 可減輕學習負擔

2. **忘記了** ❌
   - 重置該股票的學習進度
   - 1 分鐘後再次出現

3. **記得** ✓
   - 學習進度 +1
   - 間隔時間拉長 (1分鐘 → 10分鐘 → 1天 → 更久)

## 🧠 SRS 演算法說明

採用間隔重複系統 (Spaced Repetition)：
- 第 1 次答對：1 分鐘後複習
- 第 2 次答對：10 分鐘後複習
- 第 3 次答對：1 天後複習
- 之後每次：間隔時間 × 2.5

## 💾 資料儲存

- 使用瀏覽器 `localStorage` 自動儲存進度
- 關閉瀏覽器後再開啟，進度會保留
- 如需重置：開啟瀏覽終端 (F12) 輸入 `localStorage.clear()`
