## 🎯 資料品質標準

**完整資料應包含**：
1. ✅ 股票代號（id）
2. ✅ 公司名稱（name）
3. ✅ 產業別（industry）
4. ✅ 上市/上櫃（market）
5. ✅ 一句話描述（description）
6. ⭐ 量化指標（keyMetric）- 選填但建議有

## 🔄 如何擴充資料庫

如需新增更多公司：
1. 編輯 `stocks_top100.json`
2. 按照現有格式添加新公司
3. 確保包含所有必填欄位
4. 用 `stats.html` 確認資料完整性
